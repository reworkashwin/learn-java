But if you try to execute these commands straight away, the unit testing is going to fail because during the packaging of the application, the maven will try to perform some basic unit testing and during this unit testing process, it will try to connect to the database and if the database is not available, the whole packaging process is going to fail. Since right now I'm not worrying about anything around the unit testing, what I'll do is, after this install, I'm going to mention space followed by hyphen D maven dot test dot skip is equal to true. By using this property, I'm telling to the maven, don't perform any unit testing straight away, do the compilation and packaging of my application code.
