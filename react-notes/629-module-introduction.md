# Module Introduction — Practice Project

## Introduction

You've learned the fundamentals: components, JSX, props, composition, and data flow. Now it's time to **practice**. This module is a hands-on exercise where you'll build a small demo application from scratch, applying everything you've learned so far.

---

## Concept 1: What This Section Is About

### 🧠 What is it?

This is a **practice project** — not a lecture-heavy section. The goal is to take a starting project and transform it into a working application by applying the component concepts you've learned.

### ❓ Why do we need it?

Reading about components and watching someone build them is one thing. Actually doing it yourself — making decisions about what to extract, how to pass data, where to split — is where real learning happens.

---

## Concept 2: What You'll Practice

### ⚙️ The Tasks

1. **Output dynamic data** — Take an array of data objects and render them on screen
2. **Identify components** — Analyze the existing JSX and figure out what could be extracted into separate components
3. **Create and use custom components** — Build new component files, use props to pass data, and compose them together

### 💡 Insight

These are the exact skills you'll use in every React project: analyzing a UI, breaking it into components, wiring them together with props. The more you practice now, the more natural it becomes.

---

## ✅ Key Takeaways

- This section is about **doing**, not just watching
- You'll practice: outputting data, identifying components, creating components, and using props
- Try to solve the tasks on your own before looking at the solution
- These fundamentals — components and props — are the building blocks for everything that comes next
