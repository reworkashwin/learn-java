# Starting Project

## Introduction

To explore JavaScript fundamentals, we're using a simple CodeSandbox project — a basic web project with an HTML file and some assets, but **no JavaScript code yet**. This is our sandbox for experimenting.

---

## The Setup

- A basic web project hosted on **CodeSandbox**
- Contains an HTML file and some static assets
- No JavaScript files pre-configured — we'll add them ourselves

### Where to See Output

Since this section is about refreshing JavaScript knowledge (not building a visual project), most of our code will **output to the console**.

In CodeSandbox, access the console by clicking the **"Console"** tab at the bottom of the editor.

---

## What We'll Do Here

This project serves as a playground for:

- Writing and testing JavaScript code snippets
- Exploring core syntax and modern features
- Getting comfortable with the JavaScript concepts needed for React

---

## ✅ Key Takeaways

- The starting project is intentionally minimal — just HTML and assets
- We'll add JavaScript files and write code that logs output to the console
- The focus is on learning and experimenting, not building a UI

---

## 💡 Pro Tip

Follow along by writing the code yourself in the CodeSandbox project. Don't just read — **type it out**. Muscle memory is a real thing in programming, and it helps concepts stick.
