# Project Overview

## Introduction

Before diving into the advanced patterns, let's get the demo project up and running. Every great learning experience starts with a solid foundation — and this section is no different. You'll set up the project that will serve as the playground for exploring Compound Components, Render Props, and more.

---

## Concept 1: Setting Up the Demo Project

### 🧠 What is it?

The demo project is a pre-configured React application with basic files and styling already in place. It provides the skeleton you'll fill in with logic as you learn each pattern throughout the section.

### ❓ Why do we need it?

Starting from scratch every time wastes valuable learning time. The starter project gives you a clean, consistent environment so you can focus entirely on the patterns and concepts — not boilerplate setup.

### ⚙️ How it works

1. **Download the project** — available both as a local project and a CodeSandbox link
2. **If using the local setup:**
   - Run `npm install` to install all dependencies
   - Run `npm run dev` to start the development server
3. **If using CodeSandbox:** No installation needed — it works out of the box
4. Once running, you should see a basic page rendered in the browser

### 🧪 Example

After starting the dev server, the page should display a basic layout. The folder structure contains some initial files that are mostly empty — these are the files you'll populate with logic as you progress through the section.

```
src/
├── components/    # Where your pattern components will live
├── App.jsx        # Main application component
├── index.css      # Pre-built CSS styles
└── ...
```

### 💡 Insight

The starter project intentionally comes with **pre-built CSS classes** (like `accordion`, `accordion-item`, `searchable-list`, etc.). This means you won't have to worry about styling — you can focus purely on understanding the React patterns. The styling is there to make the final result look polished without you having to write a single line of CSS.

---

## ✅ Key Takeaways

- The demo project is a lightweight React app with pre-configured files and styling
- Use `npm install` + `npm run dev` for local development
- CodeSandbox requires no setup
- The project's folder structure is simple — you'll add logic to it as you learn each pattern

## 💡 Pro Tips

- Have the project open and running alongside the lessons — code along for the best learning experience
- Don't worry about the CSS — it's already been written for you
- Familiarize yourself with the file structure before moving on so you know where things will go
