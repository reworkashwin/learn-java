# The Starting Project for Debugging

## Introduction

This section uses a slightly modified version of the **Investment Calculator** project from earlier in the course. The code has been intentionally tweaked to include some errors and bugs that we'll systematically find and fix.

---

## What's in the Starting Project

The project is a simple React app with:
- A few components managing state and computed values
- User inputs for investment parameters
- A results table that shows calculated investment data

The key thing to know: **the project has bugs**. Some are obvious (error messages), and some are subtle (wrong calculations). This is intentional — we'll use them as learning opportunities.

---

## Don't Worry If You Skipped the Practice Section

Even if you didn't build the Investment Calculator yourself in the earlier practice section:
1. You need the **attached project** anyway (it's been modified with bugs)
2. The app is straightforward enough to follow along without prior context
3. The focus is on **debugging techniques**, not the app's business logic

---

## ✅ Key Takeaway

Download the attached starting project. The bugs in it are deliberate and designed to teach you specific debugging techniques. The project itself is simple — the real lesson is in the process of finding and fixing errors.
