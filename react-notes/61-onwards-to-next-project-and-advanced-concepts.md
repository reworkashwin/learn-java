# Onwards To The Next Project & Advanced Concepts

## Introduction

You've just completed your first React project, diving deep into components, props, and JSX. Now it's time to level up — we're building a **Tic-Tac-Toe game** from scratch!

This isn't just a fun project. It's a vehicle for exploring more advanced React patterns and concepts that every developer needs to master.

---

## What You'll Learn in This Project

This Tic-Tac-Toe game will cover:

- **Revisiting essential concepts** — components, props, state, and JSX in action
- **State management patterns** — deriving state, lifting state up, and managing complex state
- **Outputting list data** — rendering dynamic lists and multi-dimensional arrays
- **Computed/derived values** — calculating values from state instead of storing redundant data
- **Advanced state patterns** — immutable updates, function-form state updates, and more

By the end, you'll have a **solid, practical understanding** of how these concepts work together in a real application.

---

## Getting Started with the Project

### Setting Up Locally

1. Download the starting project ZIP file (attached to the lecture)
2. Extract the ZIP file
3. Run `npm install` to install all dependencies
4. Run `npm run dev` to start the development server
5. Open the URL shown in the terminal to see your app

### Using CodeSandbox

If you prefer browser-based development, use the CodeSandbox link — no setup needed. CodeSandbox handles `npm install` and `npm run dev` automatically.
