# Module Introduction — React Basics & Components

## Introduction

Welcome to the heart of React! In the first section, we got a high-level overview of what React is and why we'd want to use it. Now it's time to roll up our sleeves and dive into the **core features** you need to build any React application. This section is all about **component-driven user interfaces** — the fundamental building block of every React app.

---

### Concept 1: What This Section Covers

#### 🧠 What is it?

This section covers all the **base features** of React — the core syntax and concepts that you'll use in every single React project you ever build.

#### ⚙️ How it works

Specifically, you'll learn:

- **JSX** — A special syntax that lets you write HTML-like code inside JavaScript
- **Components** — What they are, how to build your own, how to use and compose them
- **Working with data** — How to output data, pass data around, and update the UI when data changes

#### 💡 Insight

Most web applications aren't about showing static text — they're about **outputting dynamic data** and **reacting to user interactions**. That's exactly what this section prepares you for.

---

### Concept 2: The Project We'll Build

#### 🧠 What is it?

Throughout this and the next few sections, you'll build an **Expense Tracker** application — a fully functional React app that lets you:

- View expenses organized by year
- Filter expenses by year (with the UI updating automatically)
- Add new expenses with a title, amount, and date
- See a chart that updates dynamically based on your data

#### ❓ Why do we need it?

This project might look simple on the surface, but it covers **all the important key concepts** you'll use repeatedly in any React app. By the end, you'll have hands-on experience with every core React feature.

#### 💡 Insight

It's intentionally not a massive project — the goal is to learn React fundamentals through a focused, manageable application. Once you finish it, you'll have the building blocks for much larger apps.

---

## ✅ Key Takeaways

- This section is the foundation — everything you learn here applies to every React project
- You'll learn JSX, components, and data handling
- The Expense Tracker project will be your learning vehicle
- It's built step-by-step, concept by concept, so every piece builds on the last
- The project spans multiple sections because React has many interconnected features
