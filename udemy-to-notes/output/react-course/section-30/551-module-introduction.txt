This bonus course section is about replacing Redux, this application-wide state management system, which you might be using in your React app, with React's Context API and React Hooks. And it's totally optional. You don't have to replace Redux, there's nothing wrong with Redux but if you do replace it, of course you don't need to add that extra third party library to your application anymore. In addition, this course section hopefully is just interesting by itself because we're going to dive a bit deeper into React's Context API and React Hooks, and it might be interesting to see what you can do with React Hooks and with the Context API, because we can get creative with those features and in this course section, we are going to get creative with them.
