Like all developers, as a React developer, you will eventually write code that contains errors and bugs, because we all do. And therefore in this course section you will learn how to efficiently debug your React apps, so how to find and fix errors. And we're going to get started by taking a closer look at those error messages React sometimes shows you, so that you can make sense of these error messages, and so that they actually help you find and fix errors. But we're not going to stop there, because some errors don't actually produce error messages. Instead, unfortunately, as a developer, you sometimes also write code that leads to logical errors, and therefore in this section you will also learn how you can find and fix such errors with help of the browser developer tools and the browser debugger. In addition, in this section, you will also learn about React's Strict Mode, how you can enable that mode, and why you might want to do that. And last but not least, we'll also take a look at the React DevTools, and you will learn how to install these tools and what you can do with them.
